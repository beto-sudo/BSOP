## ¿Qué es y para qué sirve?

Es el gestor de **tareas y pendientes** de DILESA: qué hay que hacer, quién es
responsable, en qué estado va y para cuándo. Las tareas pueden quedar ligadas a
una junta en curso.

## Cómo llegar

**Sidebar → DILESA → Administración → Tareas.**

## La tabla

Cada renglón es una tarea: **Tarea** (título + departamento/descripción),
**Prioridad** (Urgente / Alta / Media / Baja), **Estado**, **Avance** (barra
0–100%), **Responsable**, **Compromiso** (fecha) y **Días** (cuánto lleva). Abajo,
un contador y un interruptor para **ocultar/mostrar completadas**.

## Lo que puedes hacer

- **Buscar / filtrar** — por texto, **Estado**, **Prioridad**, **Asignado a** y
  **Departamento**.
- **Nueva tarea** — capturas título, prioridad, responsable, fecha de compromiso y
  avance. Si hay una junta **en curso**, la tarea se liga sola a ella.
- **Abrir una tarea** — clic en el renglón para ver/editar y agregar **avances**
  (comentarios con fecha y autor).
- **Completar rápido** (✓) y **avance/estado inline** — disponibles para
  Administración / Dirección.

## Estados

| Estado          | Significa                                    |
| --------------- | -------------------------------------------- |
| **Pendiente**   | Aún no se inicia.                            |
| **En progreso** | En ejecución.                                |
| **Bloqueado**   | Detenida, espera algo externo (pide motivo). |
| **Completado**  | Terminada y cerrada (avance 100%).           |
| **Cancelado**   | Descartada.                                  |

## Preguntas frecuentes

**No veo todas las tareas de la empresa.**
Si no eres Administración/Dirección, ves solo las tareas donde eres
**responsable, quien la asignó o quien la creó**. Es por diseño.

**¿Quién puede cambiar el estado o el avance?**
La edición rápida (estado/avance inline, completar) es para Administración/
Dirección o quien la asignó. Cualquiera puede agregar **avances** (comentarios).

**Pasé una tarea a "Bloqueado" y me pidió un motivo.**
Sí: bloquear exige escribir por qué, para que quede claro qué la detiene.

## Si algo no cuadra

Si una tarea no se ligó a la junta esperada, revisa que la junta esté **en curso**
al momento de crearla — solo entonces se liga automáticamente.
